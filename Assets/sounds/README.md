# Sound Assets

The game looks for the files below at runtime and gracefully skips any
that are missing — it builds and runs perfectly with this folder empty.
Drop in your own files with these exact names to enable audio:

| File                     | Used for                              |
|--------------------------|-----------------------------------------|
| `menu_click.wav`         | Menu and garage navigation clicks       |
| `countdown_beep.wav`     | Each countdown tick (3, 2, 1)           |
| `go.wav`                 | The "GO!" countdown moment              |
| `engine_loop.wav`        | Engine sound while accelerating         |
| `win.wav`                | Played when the player finishes a race  |
| `nitro.wav`              | Played when nitro boost activates       |
| `lap.wav`                | Played each time a lap is completed     |
| `background_music.ogg`   | Looping race background music           |

Free, royalty-free sounds matching these names can be found on sites
like freesound.org, pixabay.com/sound-effects, or opengameart.org.

`.wav` is recommended for short sound effects and `.ogg` for the
background music track.
